**1. Demo 简介**

* **是什么：** 一个对话式陶瓷定制 Web 应用，用户用自然语言描述想要的陶瓷摆件，AI 自动生成 3D 造型方案，直连景德镇/德化等产区的陶瓷工作室完成烧制配送。

* **面向谁：**
  - 核心用户：25–40 岁追求个性化家居/礼物的年轻人，尤其是新中式、治愈系、二次元偏好者
  - 送礼场景用户：想给亲友送有故事的礼物但缺乏设计能力的人群
  - 供给侧：景德镇、德化、宜兴等产区的中小陶瓷工作室与非遗手艺人

* **主要功能：**
  - **对话式设计台**：输入自然语言描述，AI 自动解析造型、釉色、尺寸、用途等关键词，30 秒生成 3 个可烧制的 3D 方案
  - **方案微调**：支持自然语言局部调整（如"把兔子耳朵再长一点点"），所见即所得
  - **智能派单与下单**：按工艺、产能、地域匹配最合适的工作室，一键下单，结构化工单直达师傅手中

**2. Demo 创作思路**

* **灵感来源：** 朋友想给妈妈生日送一只"像她养的那只橘猫坐在月亮上"的陶瓷摆件，逛遍淘宝没找到，问了三家工作室都报价上千还要等两周打样。这个经历让我意识到——自然语言到造型的"翻译"，正是大模型最擅长的事，而陶瓷行业又恰好缺这一环。

* **想解决的问题：**
  - **需求侧痛点：** 标品撞款严重，叙事性造型几乎搜不到；找工作室定制平均要 5–8 轮文字+草图来回，效果要等打样才能看到；付了定金才发现跑偏，沉没成本高
  - **供给侧痛点：** 中小工作室和非遗手艺人接单依赖熟人和私域，长尾需求接不住；师傅 60% 以上时间花在沟通上，真正拉坯烧窑的时间被严重挤压；缺乏稳定的需求来源，产能闲置和爆单交替

* **为什么做这个方向：** 我判断陶瓷定制从"高门槛非标服务"变成"所见即所得的轻量消费"是一个明确的趋势。客单价 200–800 元区间是目前严重缺供给的市场，而 AI 恰好能解决"需求理解"这个最大的瓶颈。同时，景德镇、德化有大量手艺扎实但不擅长线上运营的工作室，陶语能给传统陶瓷手艺装一条数字化的需求管道，这也是我加选社会公益标签的原因。
