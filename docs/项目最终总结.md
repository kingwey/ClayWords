## 💡 技术亮点

### 1. 向量检索优化

```sql
CREATE INDEX ON design_templates 
USING ivfflat (embedding vector_cosine_ops) 
WITH (lists = 100);
```
**性能**: 1536 维向量检索 < 10ms

### 2. 双写策略

```python
# PostgreSQL 持久化
task = await db.add(task)
# Redis Stream 入队
await redis.xadd("tasks_stream", {...})
```
**优势**: 数据不丢失 + 高性能

### 3. SSE 断线重连

```python
# 客户端发送 Last-Event-ID
last_id = request.headers.get("Last-Event-ID")
# 服务端回放历史
if last_id:
    history = await redis.xrange("events", last_id)
```
**优势**: 30 秒内断网不丢消息

### 4. 幂等性键防重

```python
idempotency_key = f"payment_callback_{trade_no}"
if await redis.exists(idempotency_key):
    return "success"
await redis.setex(idempotency_key, 86400, "processed")
```
**优势**: 支付回调防重放

### 5. 多层防护体系

```
用户输入 → Pydantic → 速率限制 → Prompt 防护 
→ JWT → 参数化查询 → 字段加密 → 数据库
```
**优势**: 纵深防御

### 6. Helm 一键部署

```bash
helm install claywords ./helm/claywords \
  --namespace production \
  -f values-production.yaml
```
**优势**: 声明式配置 + 版本化 + 一键回滚
