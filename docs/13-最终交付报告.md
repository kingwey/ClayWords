## 二、Hunyuan3D 接入详情

### 2.1 功能概述

集成腾讯云混元 3D API，支持从文本/图片生成 3D 模型（GLB/OBJ 格式）。

**核心特性**:
- ✅ 异步任务管理（提交秒级返回，后台轮询）
- ✅ 支持文本 Prompt 和图片输入
- ✅ 支持 Model 3.0 / 3.1
- ✅ 返回 GLB + OBJ + 预览图
- ✅ 超时保护（15 分钟）
- ✅ 功能开关（ENABLE_HUNYUAN3D）

### 2.2 代码实现

| 文件 | 行数 | 说明 |
|------|------|------|
| `app/core/config.py` | +7 | 新增 5 个配置项 |
| `app/services/hunyuan3d/schemas.py` | 79 | 请求/响应数据模型 + 辅助方法 |
| `app/services/hunyuan3d/client.py` | 125 | API 客户端（submit + query） |
| `app/services/hunyuan3d/worker.py` | 235 | 后台轮询 worker |
| `app/api/hunyuan3d.py` | 115 | REST 端点 |
| `app/main.py` | +2 | 路由注册 |
| `scripts/test_hunyuan3d.py` | 110 | 测试脚本 |
| `backend/.env` | +6 | 环境配置 |

**总计**: 617 行新代码 + 15 行修改

### 2.3 架构设计

```
┌─────────────┐
│ 用户提交请求 │ POST /api/v1/hunyuan3d/submit
└──────┬──────┘
       │
       ▼
┌─────────────────────────────────────┐
│  ClayWords Backend                  │
│  - 调用 Hunyuan3D submit 接口       │
│  - 创建 Task 记录 (task_id, job_id) │
│  - 返回 task_id 给前端              │
└──────┬──────────────────────────────┘
       │
       ▼
┌─────────────────────────────────────┐
│  ARQ 后台 Worker (每 5 秒轮询)      │
│  - 调用 Hunyuan3D query 接口        │
│  - 任务完成 → 下载 3D 文件到 MinIO  │
│  - 更新 Task 状态                   │
│  - 通过 SSE 推送进度给前端          │
└─────────────────────────────────────┘
```

### 2.4 API 响应格式适配

发现并修复了文档与实际的差异：

**文档说明** (错误):
```json
{
  "Status": "completed",
  "ResultUrl": "https://..."
}
```

**实际返回**:
```json
{
  "Response": {
    "Status": "DONE",
    "ResultFile3Ds": [
      {"Type": "GLB", "Url": "https://...", "PreviewImageUrl": "..."},
      {"Type": "OBJ", "Url": "https://..."}
    ],
    "ResultCreditConsumed": 20
  }
}
```

**已适配**:
- ✅ 解析 `Response` 包装层
- ✅ 处理 `DONE` / `RUN` / `FAILED` 状态
- ✅ 支持 `ResultFile3Ds` 数组
- ✅ 提供 `get_glb_url()` / `get_obj_url()` 辅助方法
