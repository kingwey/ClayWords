# ClayWords MVP 上线 Sprint 计划

> 最快上线路径: 15-20 天  
> 目标: 核心功能可用，支持完整用户流程  
> 当前状态: 后端 95% 就绪，前端 0%

---

## 🎯 Sprint 目标

**上线 MVP 版本，支持核心流程**：
1. 用户登录注册
2. 文字描述需求（暂无 3D 生成）
3. 创建订单
4. 支付宝支付
5. 工作室接单制作
6. 物流发货
7. 确认收货

## 🚫 MVP 不包含的功能

以下功能延后到 v1.1:

1. **3D 模型生成** (Phase Q3)
   - MVP 使用固定的 3 个方案图片
   - v1.1 集成 Hunyuan3D

2. **管理后台**
   - MVP 使用数据库直接操作
   - v1.1 开发管理后台

3. **微信支付**
   - MVP 仅支持支付宝
   - v1.1 集成微信支付

4. **实时物流**
   - MVP 仅显示物流单号
   - v1.1 集成快递 API

5. **用户评价**
   - MVP 无评价功能
   - v1.1 增加评价系统

## 🚨 风险管理

### 高风险

| 风险 | 影响 | 概率 | 缓解措施 |
|------|------|------|---------|
| 前端开发延期 | 推迟上线 | 中 | 2-3 人并行开发 |
| 支付宝生产配置 | 无法支付 | 低 | 提前申请生产密钥 |
| 服务器资源不足 | 性能问题 | 低 | HPA 自动扩容 |

### 中风险

| 风险 | 影响 | 概率 | 缓解措施 |
|------|------|------|---------|
| Bug 修复耗时 | 推迟上线 | 中 | 预留 3 天 Buffer |
| DNS 解析延迟 | 部分用户无法访问 | 低 | 提前配置 DNS |
