## Phase 0 · 基础设施部署（当前优先级 P0）

> 目标：配置实际 K8s 集群和生产环境  
> 预计时间：2-3 天  
> 负责人：OPS

### 0.1 K8s 集群配置

- [ ] **0.1.1** [P0/OPS] 购买云服务器资源
  - 阿里云 ACK 或腾讯云 TKE
  - 3 个 Master 节点（4C8G）
  - 5 个 Worker 节点（8C16G）
  - 1 个 GPU 节点（可选，Phase Q3 需要）
  - 验收：`kubectl get nodes` 显示 8 个节点 Ready

- [ ] **0.1.2** [P0/OPS] 配置 kubectl 本地访问
  - 下载 kubeconfig
  - 配置 kubectl context
  - 验收：`kubectl cluster-info` 成功

- [ ] **0.1.3** [P0/OPS] 安装必要组件
  - Nginx Ingress Controller
  - cert-manager (TLS 证书管理)
  - Prometheus Operator
  - 验收：`kubectl get pods -A` 核心组件 Running

### 0.2 域名和证书

- [ ] **0.2.1** [P0/OPS] 购买域名
  - claywords.com
  - 可选：claywords.cn
  - 验收：域名控制面板可访问

- [ ] **0.2.2** [P0/OPS] DNS 配置
  - A 记录：api.claywords.com → K8s Ingress IP
  - A 记录：claywords.com → K8s Ingress IP
  - A 记录：www.claywords.com → K8s Ingress IP
  - 验收：`nslookup api.claywords.com` 解析正确

- [ ] **0.2.3** [P0/OPS] TLS 证书配置
  - cert-manager ClusterIssuer（Let's Encrypt）
  - 自动签发和续期
  - 验收：浏览器访问 https://api.claywords.com 显示绿锁

### 0.3 密钥管理

- [ ] **0.3.1** [P0/SEC] 生成生产密钥
  - CRYPTO_PEPPER (32 字节)
  - JWT_SECRET_KEY (64 字节)
  - DATABASE_PASSWORD (32 字符)
  - MINIO_SECRET_KEY (32 字符)
  - 验收：密钥保存到 Vault 或安全位置

- [ ] **0.3.2** [P0/OPS] 创建 K8s Secrets
  - `kubectl create secret generic claywords-secrets`
  - 包含所有生产密钥
  - 验收：`kubectl get secret claywords-secrets -n production` 存在

### 0.4 部署到生产

- [ ] **0.4.1** [P0/OPS] 部署到 Staging
  - 执行 `helm install claywords-staging`
  - 验收：所有 Pod Running

- [ ] **0.4.2** [P0/OPS] Staging 烟雾测试
  - 健康检查：`/health`
  - API 测试：创建订单流程
  - 验收：核心流程通过

- [ ] **0.4.3** [P0/OPS] 部署到 Production
  - 执行 `bash scripts/deploy_production.sh`
  - 验收：生产环境可访问

- [ ] **0.4.4** [P0/OPS] 生产烟雾测试
  - 健康检查
  - Prometheus 指标
  - 日志输出正常
  - 验收：3 个 Backend Pod + 2 个 Worker Pod Running
